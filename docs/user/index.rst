User Guide
==========




Beelink sitemizin Anasayfa görünümü alttadır.

.. figure:: tuncay/anasayfa.PNG
      :alt: map to buried treasure
      :figclass: align-center

      Beelink anasayfası


Beelink uygulaması Python üzerinde Flask framework kütüphanesi kullanılarak hazırlanmıştır. Arayüz için bootstrap ve javascript kütüphaneleri kullanılmıştır.

|

Anasayfaya bağlantılı olan sayfalar:

1. Kayıt olma ve giriş yapma
2. Zaman Tüneli Sayfası
3. Profil bilgileri
4. Arkadaş ilişkileri
5. Mesajlaşma



3 farklı tasarım template kullanılmıştır, en üst düzey teknolojiler kullanılıp, Kullanıcı açısından zor ve karışık olmaması hedeflenmiştir.
